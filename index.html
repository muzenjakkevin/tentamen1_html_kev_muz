SEO och nyckelord

Det är viktigt att använda sina nyckelord på headern där webbplatsen representeras,
där din beskrivande text är om sakerna. Men även på alt atributer ifall bilderna
inte skulle laddas på webbplatsen. Om man vill ranka högt på google skall man använda sig
utav ord som passar bäst till din hemsida, och de ord som söks mest utav. Fall ens hemsida
handlar om lamborghini, skall nyckelorden vara något om lamborghini eller bara lamborghini,
för att få som mest träffar till din hemsida.

Responsiv webbdesign

Responsiv webbdesign innebär att webbsidan skall vara anpassad för olika skärmstorlekar,
utan några problem. Responsiv webbdesign använder sig bara utav HTML & CSS, det behöver inte JavaScript.
För att göra sidan responsiv kan man använda sig utav dessa två exempel, meta viewport taggen
och media queries. Breakpoints är punkter där CSS bryter vid en viss pixelpunkt till en annan.
Till exempel fall man vill gå från desktop till mobil, vill man att viewporten skall anpassas till
användarens enhet. Då har man satt brytpunkter för den mobila enheten så att de skall se anorlunda ut.
